# Guide how to test the backend

### Clone the repo
```bash
git clone https://github.com/Huerte/ShopNow.git
```

### Go to backend directory path
```bash
cd src/backend
```

### Install dependencies
```bash
pip install -r requirements.txt
```
- If error - contact me directly at mailto:huertejerald@gmail.com

### Now Run the server
```bash
uvicorn main:app --reload
```
- If error - contact me directly at mailto:huertejerald@gmail.com
